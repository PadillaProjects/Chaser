# Customizable Character Feature

## Overview
The Customizable Character feature allows players to personalize their in-game avatar. This character is the central representation of the player throughout the application, visible in game sessions and on the main menu.

## Main Menu Integration
- **Placement**: The player's customized character appears at the top of the Main Menu screen.
- **Context**: The character is displayed prominently above the list of active or available sessions (join session or see your sessions).
- **Interaction**: A **Customize** button is available near the character display.

## Customization Interface
Clicking the "Customize" button navigates the user to the Character Customization Screen. Here, players can view and modify their character's attributes.

### Customizable Attributes
Based on the profiles system, players can modify:

#### Appearance
- **Head**: Headwear, Masks.
- **Torso**: Skins, Shirts, Armor.
- **Bottom**: Pants.
- **Extremities**: Arms, Boots.

#### Animations
- Running style
- Celebration
- Catcher release animation
- Idle animation
- Capturing animation

#### Gear & Extras
- **Gear**: Weapons, Ride (skateboard, skates, etc.).
- **Experience**: Sounds (emitted when close), Profile Backgrounds.
